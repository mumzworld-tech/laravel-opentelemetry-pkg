# Changelog

All notable changes to `mumzworld/laravel-opentelemetry` will be documented in this file.

## [1.0.0] - 2024-12-19

### Added
- Initial release of Laravel OpenTelemetry package
- TracerService for custom business logic tracing
- Complete Docker observability stack (Collector, Tempo, Grafana)
- Environment-driven configuration
- Automatic Laravel service provider registration
- Comprehensive documentation and examples
- Migration guide from custom implementations

### Features
- ✅ Easy integration with single composer require
- ✅ All settings configurable via environment variables
- ✅ Docker-ready observability stack
- ✅ Custom tracing with TracerService
- ✅ Automatic OpenTelemetry bootstrap
- ✅ Production-ready configuration options
